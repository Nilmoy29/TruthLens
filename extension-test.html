TruthLens Extension Test Page

How to Test the Extension:

  1. Load the Extension: Go to Chrome Extensions (chrome://extensions/), enable Developer mode, and click "Load unpacked" to load the extension folder.
  2. Test Popup: Click the TruthLens extension icon in the toolbar to open the popup.
  3. Test Context Menu: Right-click on selected text below to see TruthLens context menu options.
  4. Test Analysis: Use the popup to analyze text or this page URL.

Extension Features to Test:

  • ✅ Popup opens when clicking extension icon

  • ✅ "Use Page URL" button captures current page URL
  • ✅ Analysis requests are sent to the API
  • ✅ Results are displayed in the popup
  • ✅ Copy to clipboard functionality works
